这是一个文件说明的txt文件！

try_1.py是SMOTE均衡处理的数据预处理文件；
try_2.py是特征提取文件；
try_4.py是最大分类分离距离的简单的阈值筛选；
try_4_d_XGB.py是根据最大分离距离的模型预测评分来筛选；
XGBoost.py是简单的XGBoost运用；
XGBoost_CV.py是根据网格搜索得到最佳的最大树数目和较优的参数设置；
XGBoost_model.model是根据try_4_d_XGB.py得到的该方法下最佳的模型；
test_1.py，test_2.py，test_3_XGBoost.py分别是无标签的测试数据的特征提取，特征选择和模型预测。
